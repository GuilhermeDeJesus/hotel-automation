# Resumo de Refatoração

Síntese das decisões técnicas que permanecem válidas.

## Decisões

- Regras de negócio no domínio.
- Integrações externas na infraestrutura.
- Use cases como orquestradores.
- Dependências centralizadas em `dependencies.py`.

## Ganhos

- Menor acoplamento.
- Melhor testabilidade.
- Evolução incremental mais simples.
